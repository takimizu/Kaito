# 日本語
## 滝水かいと
滝水かいとは滝水兄弟の一人です。ここのコードはBot本体(Discord.pyによって作成)とWebアプリケーション(PHPによって作成)です。

### インストール関係

``` $ python -m pip install discord.py[voice]```
